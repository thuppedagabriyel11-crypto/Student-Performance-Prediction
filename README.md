# 🎓 Student Performance Prediction

## Project Overview

This project uses Machine Learning to predict student mathematics performance.

## Technologies Used

- Python
- Pandas
- Scikit-Learn
- Streamlit
- Machine Learning

## Algorithm

Random Forest Regression

## Features

- Dataset analysis
- ML model training
- Performance prediction
- Interactive web interface

## How to Run

Install requirements:

pip install -r requirements.txt

Run:

streamlit run app.py
